## Features
- 3×5 switch matrix
- Custom Arduino firmware
- USB MIDI output
- Works with FL Studio and other DAWs

## Hardware
- Arduino Nano
- 15 Tactile Switches
- 15 Diodes
- USB Cable
- Perorated borad
